# Aplicação de Serviço de E-mails

A aplicação de serviço de e-mails com sockets foi desenvolvida na linguagem Python. Assim, para executar os códigos desenvolvidos, você precisa ter o Python3 instalado. A versão recomendada é a Python 3.6.9, sendo que não foram realizados testes para outras versões.

## Instruções de execução

Para testar nossa aplicação de e-mails baseada em sockets, você precisará de 3 terminais.
Em um deles você vai rodar o *server.py*.

~~~
python3 server.py 
~~~

Em cada um dos outros dois terminais, você vai rodar o *client.py*.

~~~
python3 client.py 
~~~

O usuário não terá nenhuma interação de fato com o servidor (server.py), apenas com os dois clientes. A utilização dos clientes é bastante intuitiva. Para maiores detalhes e um exemplo prático do funcionamento, você pode assistir esse [vídeo](https://www.youtube.com/watch?v=V7175behCro&t=1s).
